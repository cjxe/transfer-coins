# Welcome to Transfer Coin's Design System

This guide will help you start working with [Transfer Coin](https://transfer-coins.vercel.app/)'s
Design System.

## Features

- ✅ >15 components
- ✅ >7 coding blocks
- ✅ Clean documentation with Storybook
- ✅ **Dark mode** support for happier eyes
- ✅ **Responsive** design for sharper eyes
- ✅ Cool **animations** and transitions
- ✅ Up-to-date dependencies _(React v18, Next.js v13, Tailwind CSS v3, Storybook v7)_
- ✅ _Almost_ **dependency-free** component library

## File structure

We grouped components into 4 main categories:

- `Action`/`Data input`
- `Data display`
- `Navigation`
- `Utils`
